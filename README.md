## ⚙️ Instalação
```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/maikonalexandre/ignite-chapter-3
```
```
# Instale as dependências
$ yarn

# Inicie a aplicação
$ yarn dev
```
